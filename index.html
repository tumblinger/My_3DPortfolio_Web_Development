The programs ...

... I have learnt during my course on 3D visualisation and gaming

3Ds Max

3d scene modeling begins with the 3d Max. I learned how to model, create realistic renderers, and process the render in Photoshop to get a better picture. I can work in 3D modeling and visualization, with materials, lighting, and modeling in size and exact accordance with drawings or photographs.

Megascans

Quixel Megascans are photorealistic high-resolution scans of real-world scenes and objects. I learned how to create small architectural forms, landscapes, and plants of any complexity, seamless textures, normal-bumps, and displays in the works, as well as set up compositional lighting.

ArchiCAD

To start modeling in 3d, you need to prepare drawings. I have learned and used ArchiCAD and AutoCAD programs for this purpose.

Illustrator

In Adobe Illustrator, I made illustrations, which I also used for animations. In this editor, we can create advertising layouts and use the capabilities of the program to create images.

Unreal Engine

I studied the Unreal Engine game engine and the animation capabilities of 3Ds Max (architectural animation). I created a game level, set up a character and animation sequences, taught the specifics of creating projects for Oculus or HTC VIVE virtual reality helmets.

ZBrush

I have mastered retopology in 3Ds Max, learned how to sculpt in ZBrush to create high-quality scans for any objects and produce complex texturing in real-time programs such as Substance Painter. I've studied all the stages of creating a game character and learned various renderings in real-time.
